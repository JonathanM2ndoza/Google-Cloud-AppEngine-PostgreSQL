== HTTP Headers
Los REQUEST deben tener los siguientes elementos en el header:

|===
|Header|Type|Structure|Description

|Json Web Token
|Security
|`+Authorization: {token}+`
|Token de acceso JWT

|===
* Enviar estos headers en los REQUEST, a menos que se indique lo contrario.
