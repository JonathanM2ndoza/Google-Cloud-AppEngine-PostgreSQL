[[overview-http-headers]]
== HTTP Headers
Los endpoints deben tener los siguientes elementos en el header del REQUEST:

|===
|Header|Type|Structure|Description

|Json Web Token
|Security
|`+Authorization: {token}+`
|Token de acceso creado con JWT (proporcionado por endpoint /api/v1/auth/signin)

|===
* Asumir que siempre van estos headers a menos que se indique explicitamente
